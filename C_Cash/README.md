## Cash (from CS50 course Problem Set 1)

#### Description
Program to decide which coins and how many of each to hand to the customer using Greedy Algorithms

#### Technologies used:
- *C*

#### Usage
```
$ ./cash
Change owed: 41
4
```
```
$ ./cash
Change owed: -41
Change owed: foo
Change owed: 41
4
```
