## Scrabble C (from CS50 course Lab 2)

#### Description 
Program to determine which of two Scrabble words is worth more.

#### Technologies used:
- *C*

#### Usage example
```
$ ./scrabble
Player 1: COMPUTER
Player 2: science
Player 1 wins!
```
